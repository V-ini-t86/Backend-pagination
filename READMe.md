# Backend-pagination

#### Langauge: Javascript

#### Database: MongoDB

#### Framework: Express

### To install dependencies

```
yarn install || npm install
```

#### To run this server

```
yarn run dev || npm run dev
```

**Paginated Url**

```
http://localhost:3000/users?page=2&limit=5
```
